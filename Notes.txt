1st vid:

install cli:
npm install -g @angular/cli

install project shell (USE BASH TERMINAL :P. none strict mode is optional):
ng new myFirstProject --no-strict

build and run project (when in project dir :P - i.e where .angular folder lives):
ng serve

dev server:
http://localhost:4200/

stop serving app:
ctrl+c

install bootstrap (nav to project folder, as this installs locally):
npm install --save bootstrap@3.4.1 (or any version you'd like. this one was needed for max's course :P)

add bootstrap styles:
angular.json > "styles:["node_modules/bootstrap/dist/css/bootstrap.min.css", "plus whatever else was here!"]" 


-------------

create module:
ng generate module moduleName

-------------

create componenet:
ng generate component componentName
